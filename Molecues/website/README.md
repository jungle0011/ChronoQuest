# Molescules Website

Landing page, privacy policy, data deletion request, and admin panel for Molescules.

## Deploy

- Push to GitHub
- Import on Vercel
- Set environment variables (if any)
- Deploy

## Admin Panel

- Visit `/boss.html`
- Generate a JWT using `generate-jwt.html` or use `boss123` for demo
- Login to view users, communities, and perform admin actions.
